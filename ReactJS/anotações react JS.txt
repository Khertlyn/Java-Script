permite programar em HTML E javaScript em um mesmo arquivo,  usando método DOM renderização, para passar comandos js utilizar chaves.
encapsular dentro de div(s)
Ex. Estrutura padrão
import React, {fragment} from "react"
import ReactDOM from "react-dom"
import "./styles.css"

const element = 'Digital Innovation'
const element2 = <h1> Olá mundo </h1>

function App() {
	return (
	<div>
	  {element}
	  {element2}
	</div>
	)
}
const rootElement = document.getElementById("root")
ReactDOM.render(<app />, rootElement)
